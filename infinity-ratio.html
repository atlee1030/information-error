Infinity Ratio

The previous data examined each province's recycling and incineration rates, identifying which materials are infinitely recyclable and which undergo limited recycling (downcycling). Based on this data, we calculated the average recycling rate for metals and glass and used this value to create an Infinity Ratio metric for each province.
